Register to issue maternity exemption certificates

Use this service to register to issue maternity exemption certificates on behalf of new or expectant mothers.

You must be a midwife, practice nurse or health visitor registered with the NMC to create an account.

What you'll need

To help us verify your professional identity, we'll ask for your:

  • last name
  • date of birth
  • NMC PIN
  • NHSmail address and password
Information:

Registration takes less than 5 minutes.

I do not have an NHSmail account

If you're working in a Trust or organisation that uses NHSmail, you can contact your local administrator or IT helpdesk to set up an account.

I'm a GP with a GMC reference number

You can still register to use this service.

To register, email nhsbsa.healthcosts@nhs.net and include your:

  • name
  • GMC reference number
  • NHSmail address

Once we’ve received this information, we’ll email you to confirm you are registered. You will then be able to use the online service.
